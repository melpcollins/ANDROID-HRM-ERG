# Use Case 06: Long Ride Mobile UX

## Goal
Support uninterrupted longer indoor rides with minimal phone interaction.

## Behaviors Implemented
- Android screen stays awake while app is open (no lock screen interruption).
- Device cards collapse after successful connect for a cleaner workout view.
- Compact red/green connection status pills remain visible.
- Sections can be re-opened by tapping the status pill when detail actions are needed.

## Expected Outcome
- Rider can complete long sessions (for example 60+ minutes) without waking or unlocking the phone repeatedly.
