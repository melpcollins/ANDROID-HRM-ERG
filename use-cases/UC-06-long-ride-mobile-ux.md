# Use Case 06: Long Ride Mobile UX

## Goal
Support uninterrupted longer indoor rides with minimal phone interaction.

## Behaviors Implemented
- Android screen stays awake while app is open (no lock screen interruption).
- Device cards keep scan and reconnect actions available from the main screen.
- Workout mode selection, setup, live status, and summary remain on a single screen.
- Stop control stays visible during a running ride.

## Expected Outcome
- Rider can complete long sessions, pause/recover from device issues, and review the end-of-ride summary without navigating away from the workout screen.
